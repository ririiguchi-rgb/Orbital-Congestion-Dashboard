# Data

`satcat.csv` is a real CelesTrak SATCAT snapshot downloaded from:

https://celestrak.org/pub/satcat.csv

Field definitions are documented at:

https://celestrak.org/satcat/satcat-format.php

The master notebook can refresh the snapshot. The raw catalogue includes historical and current records. The analysis notebook applies explicit filters for current Earth-centred Low Earth Orbit records with complete required orbital fields.
